# NKCPS Elementary Zone Finder
Shiny web application to determine which elementary school zone a student lives in. This application is intended to help kindergarten registrants determine the proper elementary school to register for.

For the 2022-2023 school year, NKCPS will open Quinton Elementary School at the northern end of the county. As such, the elementary school zones have been redrawn. This application allows anyone to enter their address to see which elementary school they will be attending. If the address is not in one of the three NKCPS elementary zones, the application notifies the user that their address is not found in NKCPS.
